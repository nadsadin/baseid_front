Приложения
В разделе "Приложения" можно выбрать внешние и внутренние сервисы, приложения и сайты деловой тематики (на которых настроена авторизация при помощи BaseID). Приложения могут быть платными, бесплатными и с тестовым периодом бесплатной эксплуатации.
BaseID Аутентификация
Сервис двухфакторной аутентификации для проверки и авторизации физических и юридических лиц. Предоставляются для всех тарифов бесплатно.
Данное приложение отключить невозможно
Бесплатно
BaseID Контрактирование
Сервис контрактирования для создания, совместного редактирования и подписания любых документов при помощи электронной подписи. Предоставляются для всех тарифов бесплатно и имеет ограничение в 3 документа в месяц.
Лимит приложения: 3 документа в месяц
Бесплатно
